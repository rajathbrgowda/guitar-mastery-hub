# Guitar Mastery Hub

A fullstack practice tracker and learning companion for guitar learners.

## Stack
- **Frontend:** Vite + React + TypeScript + MUI (Helix theme)
- **Backend:** Express + TypeScript
- **Database:** Supabase (PostgreSQL + Auth)
- **Deploy:** Vercel (client) + Render (server)

## Project Management
The Kanban board lives at `.kanban/` — start the board server before working:
```bash
cd .kanban && npm start
# Open http://localhost:4444
```
